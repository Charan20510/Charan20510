## 🚀 Featured Projects

### 🎮 Bobby Carrot — RL Game Agent
> *Python · PyTorch · PPO · Rainbow DQN · ICM · MCTS*

An RL agent trained via structured curriculum learning across **25 puzzle levels** that achieves **zero-shot generalisation on 5 unseen levels**.

- Custom Gym-style environment with **dense reward shaping**, BFS distance signals & crumble-tile penalty logic
- PPO with a **19-channel CNN encoder** + 5-phase anti-forgetting curriculum
- **Rainbow DQN** for value stability, **ICM** for curiosity-driven exploration, **MCTS** at test time using PPO value head as a learned leaf evaluator

[![View Repo](https://img.shields.io/badge/View%20Repo-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/Charan20510)
